# Build Receipt - exotic-thinking

Prepared as a public, transferable skill package on 2026-07-07.

The source package was a codebase-specific live-wiring reference. This public
package keeps the transferable mechanism: eight structural reasoning lenses and
their guardrails.

## Removed

- Codebase-specific dispatch paths and drift checks.
- Per-tool live output contracts tied to one implementation.
- Runtime/tool availability claims.

## Kept

- Abductive, negative-space, predictive, asymmetric-belief,
  external-memory-binding, concurrent, split-level, and stigmergic lenses.
- A self-test that ensures all lenses are present and project-specific tokens do
  not reappear.
